TV Database Help

Getting started
Welcome to TV Database! This help section is intended to guide you through all the features of the application, so that you can then go on by yourself in this wonderful world that is the realm of quality television.

To start off, we'll be looking at the overall process of how you and the app can work together, as well as the purpose of it all.

 What is TV Database?
The application TV Database is an episode tracker desktop program designed by BogdanBBA in C#. It uses the API (application programming interface) offered by thetvdb.com in order to connect to their huge database and update information about the TV series that you've selected in your app. You can then mark new episodes each time you watch one, allowing you not only easy tracking, but statistics that can be generated by your usage.
Select a new series → Update from online database → Watch TV and update episodes

As of now, TV Database is intended for a single user/installation and for Windows PC only. It does not collect user information of any kind.

 Attention! TV Database is not associated with thetvdb.com and abuse of their service may harm us all. Use with caution and restraint.
