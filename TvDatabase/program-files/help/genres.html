Genres

Overview
These are the stylistic categories that TV series fall into, and are useful in helping viewers discover new, similar series that they might enjoy. If you like CSI: Crime Scene Investigation, you're likely to also want to watch NCIS, which are both categorized as crime and drama.

In the application, the genres are indexed in a list in the database, in order to avoid repetitions. Genres are referenced by series (more than one genre per series is normal); a new one is added only when it is needed, and deleted when no longer referenced.

 Unlike other data types, which are downloaded from thetvdb.com in separate data structures, genres are decoded from a vertical-line separated text, and may therefore present issues if they are not properly formatted on the website.

 In-app attributes
  • id - a unique text identifier for the genre, which is generated locally, from a random sequence of characters. Used for safe reference.
  • name - the name of the genre, which is displayed in-app.
For more information on how this data is obtained during update, see the Updater section.
