Networks

Overview
These are the network companies that broadcast the series. Their scope is very limited (statistical - and even that, not yet implemented).

In the application, the networks are indexed in a list in the database, in order to avoid repetitions. Networks are referenced by series; a new one is added only when it is needed, and deleted when no longer referenced.

 In-app attributes
  • id - a unique text identifier for the network, which is obtained on update from thetvdb.com (not generated locally). Used for safe reference.
  • name - the name of the network, which is displayed in-app.
For more information on how this data is obtained during update, see the Updater section.
